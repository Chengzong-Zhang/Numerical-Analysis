# 04 LaTeX 上机报告模板

本目录收纳两种报告模板：

- `单题上机报告模板.tex`：每次只完成一道上机题时使用，配套 `单题Python代码模板.py`。
- `期末上机报告极简模板.tex`：多道题合并成一份期末上机报告时使用。
- `期末上机报告极简模板.pdf`：现有极简模板的编译预览。

## 单题模板最短使用流程

1. 复制 `单题上机报告模板.tex` 和 `单题Python代码模板.py`，按题号重命名。
2. 在 Python 文件中搜索 `必须替换`，修改方程、区间、初值、步长和精确解。
3. 在 LaTeX 文件顶部修改课程名、题目名、学号、姓名和 Python 文件名。
4. 在本目录运行：

```powershell
python ".\单题Python代码模板.py"
xelatex -interaction=nonstopmode -halt-on-error ".\单题上机报告模板.tex"
xelatex -interaction=nonstopmode -halt-on-error ".\单题上机报告模板.tex"
```

Python 程序会自动生成：

- `result_table.tex`：节点数值、精确解和误差表；
- `solution.pdf`：数值解与精确解比较图；
- `error.pdf`：绝对误差图。

LaTeX 模板会自动插入这些结果；文件尚未生成时只显示占位框，不会因此编译失败。

## 每题报告检查清单

- 问题描述完整：方程、区间、初值、步长、题目要求；
- 方法公式与代码实际采用的方法一致；
- 至少给出一张结果表或一幅图；
- 有精确解时报告误差，无精确解时做步长加密或方法对比；
- 结果分析必须包含数值现象、误差原因和最终结论；
- 提交前确认附录中的 Python 文件名正确。
