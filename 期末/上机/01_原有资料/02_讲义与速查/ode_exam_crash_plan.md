# 第五章 ODE 上机考试冲刺

## 极简结论：明天最该掌握的 5 个模板

1. **五种固定步长单步法统一模板**：Euler、改进 Euler、中点 RK2、教材 Heun、RK4。老师第五章上机第 1 题直接要求五法列表和画图比较。
2. **RKF45 自适应模板**：六个 `K`、同时算 `y4/y5`、误差合格才接受、拒绝步不更新、最后一步截到 `T`。
3. **高阶方程/方程组 + RK4 模板**：二阶方程先令 `w=[y,y']`；老师上机第 2 题就是此题型。
4. **误差和收敛阶模板**：算 `max error`，再用 `h,h/2,h/4` 和对数误差比估计阶；老师上机第 2、3 题都要求。
5. **RK4 起步 + Adams PECE 模板**：AB4 预报、AM4 校正；老师上机第 1 题明确考过。

补充防守题型：刚性线性方程组可能要求 Euler、隐式梯形、RK4 比较。老师上机第 3 题明确出现，但冲刺时只掌握线性隐式梯形矩阵公式和“显式法步长过大会发散”即可。

### 1:00-1:30：误差和收敛阶

- 用 `estimate_order` 对 Euler、改进 Euler、RK4 各跑一次。
- 产出：知道理论阶应约为 `1,2,4`，会识别代码/步长错误。
- 必背：`p=log(E(h)/E(h/2))/log(2)`。

### 1:30-2:15：RKF45 自适应

- 对照代码逐行标记：六个 `K`、`y4/y5`、`err`、接受、拒绝、调步长、最后一步。
- 把 `tol` 从 `1e-4` 改到 `1e-8`，观察接受步数增加。
- 产出：能解释为什么拒绝步不能更新 `t,y`。
- 最易错：
  - `err > tol` 时错误地更新了 `t` 或 `y`；
  - 忘记 `h=min(h,T-t)`，终点越界；
  - `err=0` 除零；
  - 向量误差没取范数；
  - 新步长未限制在 `[h_min,h_max]`；
  - `K_i` 是斜率时，组合中必须乘外层 `h`。

### 2:15-2:45：二阶方程化系统

## 考场套模板顺序

1. 先抄模板，**只改** `f,t0,y0,T,h/tol,exact`。
2. 若是二阶或高阶，先化为向量系统，再调用同一个方法。
3. 先用 RK4 小步长跑出参考结果，确认符号和数组维度。
4. 再运行题目指定方法，打印终值/表格/误差。
5. 要收敛阶就用 `h,h/2,h/4`；要自适应就额外输出 `hs`。
6. 显式法结果突然爆炸时，检查是否为刚性题或步长过大。

## 考场检查清单

- [ ] `f(t,y)` 的自变量顺序、正负号、括号正确。
