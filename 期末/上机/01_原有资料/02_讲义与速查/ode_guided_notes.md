# 数值 ODE 上机硬核短讲义

配套脚本：`ode_guided_learning.py`。先认统一输入：

```python
f, t0, y0, T, h       # 固定步长
f, t0, y0, T, h0, tol # 自适应
```

## 1. Euler

- 公式：`y[n+1] = y[n] + h*f(t[n], y[n])`
- 代码：先读 `euler_simple`，考试调用 `forward_euler`
- 易错：漏乘 `h`；把新值误用进本步右端
- 用途：最简单基线；刚性问题步长过大会发散

## 2. 改进 Euler

- 公式：`k1=f(t,y)`，预测 `yp=y+h*k1`，`k2=f(t+h,yp)`，更新 `y+h*(k1+k2)/2`
- 代码：`improved_euler`
- 易错：`k2` 必须在预测终点计算
- 用途：题目写“改进 Euler / 显式梯形 / Euler 预测校正”

## 3. 中点 RK2

- 公式：`k1=f(t,y)`，`k2=f(t+h/2,y+h*k1/2)`，更新 `y+h*k2`
- 代码：`midpoint_rk2`
- 易错：最后用的是 `k2`，不是平均值
- 用途：题目写“中点方法 / 变形 Euler”

## 4. Heun RK2

- 公式：`k1=f(t,y)`，`k2=f(t+2h/3,y+2h*k1/3)`，更新 `y+h*(k1+3k2)/4`
- 代码：`heun_rk2`
- 易错：本课程 Heun 是 `a2=2/3`，不要和别处同名公式混用
- 用途：老师 `problem1.m` 的五法比较

## 5. 经典 RK4

- 公式：四个斜率 `k1,k2,k3,k4`，权重为 `1,2,2,1`，最后除以 `6`
- 代码：`rk4`
- 易错：`k2` 用 `k1`，`k3` 用 `k2`，`k4` 用 `k3`
- 用途：默认高精度固定步长方法；给 Adams 提供起步值
- 考场只改：`f,t0,y0,T,h`

## 6. RKF45 自适应

- 公式关系：六个共享斜率 `K1,...,K6` 同时组成 `y4`、`y5`
- 误差：`err = norm(y5-y4, inf)`
- 代码：`rkf45_adaptive`
- 接受：`err<=tol` 才更新 `t,y`，并保存接受步 `h`
- 拒绝：`err>tol` 不更新 `t,y`，只缩小 `h` 后重算
- 易错：拒绝步仍前进；越过 `T`；把尝试步而非接受步存进 `hs`
- 用途：题目要求误差控制、实际步长或自适应算法
- 输出：`ts,ys,hs`，应满足 `len(hs)==len(ts)-1`

## 7. 误差与收敛阶

- 最大误差：`E(h)=max_n |y_n-y_exact(t_n)|`
- 阶数：`p=log(E(h)/E(h/2))/log(2)`
- 代码：`max_error`、`estimate_order`
- 理论：Euler 约 1 阶；三种 RK2 约 2 阶；RK4 约 4 阶
- 易错：精确解抄错、向量取错分量、刚性导致显式法不稳定

## 8. 二阶化系统

- 模板：`y''=g(t,y,y')`，令 `w=[y,y']`
- 系统：`w'=[w[1], g(t,w[0],w[1])]`
- 代码：`harmonic_system`，再调用 `rk4`
- 读取：`ys[:,0]` 是 `y`，`ys[:,1]` 是 `y'`
- 用途：老师 `problem2.m` 类型；收敛阶只看 `y` 时传 `component=0`

## 9. Adams PECE

- 起步：先用 RK4 得到 `y1,y2,y3`
- P：AB4 预测；E：算预测点函数值；C：AM4 校正；E：保存校正点函数值
- 代码：`adams_pece4`
- 易错：必须固定等步长；旧函数值下标错；没有 RK4 起步
- 用途：老师 `problem1.m` 第二问，与 RK4 比较

## 10. 老师题型映射

- 五种方法 + 图：五个固定步长函数 + `plot_results`
- 二阶 + RK4 + 阶：写系统函数 + `rk4` + `estimate_order`
- 刚性方程组：`f=A@y+g(t)`；Euler/RK4 直接调用；线性隐式梯形用 `np.linalg.solve`
- 自适应：`ts,ys,hs=rkf45_adaptive(...)`，输出接受点和接受步长
