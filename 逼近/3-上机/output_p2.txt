==============================================
  第三章上机作业2：最小二乘数据拟合
  利用例3.5.3数据
==============================================

例3.5.3 数据：m = 10 个数据点
x 范围：[0.000, 5.617]
y 范围：[1.000, 29.770]

===== 第(1)部分：多项式函数空间 P_n 中的最小二乘拟合 =====

模型：φ(x) = a_0 + a_1·x + a_2·x^2 + ... + a_n·x^n

理论背景（教材式3.5.2）：
  设计矩阵 A（m×(n+1)）：A(i,k+1) = x_i^k
  法方程组：(A^T A) c = A^T y
  均方误差：L = (1/m)||Ac - y||_2^2

--- n = 1 次多项式拟合 ---
  设计矩阵 A（10×2）条件数 cond(A) = 8.1998e+00
  法方程矩阵 A^TA 条件数 = 6.7237e+01（≈ cond(A)^2）
  均方根误差（RMS）= 2.155425
  均方误差（L）= 4.645856

--- n = 2 次多项式拟合 ---
  设计矩阵 A（10×3）条件数 cond(A) = 6.2211e+01
  法方程矩阵 A^TA 条件数 = 3.8702e+03（≈ cond(A)^2）
  均方根误差（RMS）= 0.553638
  均方误差（L）= 0.306515

--- n = 3 次多项式拟合 ---
  设计矩阵 A（10×4）条件数 cond(A) = 5.4645e+02
  法方程矩阵 A^TA 条件数 = 2.9861e+05（≈ cond(A)^2）
  均方根误差（RMS）= 0.375963
  均方误差（L）= 0.141348

--- n = 4 次多项式拟合 ---
  设计矩阵 A（10×5）条件数 cond(A) = 5.7835e+03
  法方程矩阵 A^TA 条件数 = 3.3449e+07（≈ cond(A)^2）
  均方根误差（RMS）= 0.365702
  均方误差（L）= 0.133738

===== 第(2)部分：基函数集 {1, x, cos(x), sin(x)} 拟合 =====

理论背景（教材§3.5.1 推广形式）：
  基函数：φ_0=1, φ_1=x, φ_2=cos(x), φ_3=sin(x)
  拟合模型：f(x) = a_0 + a_1·x + a_2·cos(x) + a_3·sin(x)
  线性无关性：这四个函数在数据点集上满足 Haar 条件（教材定理3.5.1）
  法方程组同样为 (A^T A)c = A^T y，只是 A 的列不再是幂函数

设计矩阵 A 的结构：
  A = [1, x_i, cos(x_i), sin(x_i)]（10×4 矩阵）
  条件数 cond(A) = 1.3830e+01

拟合系数：
  a_0（常数项 1 的系数）   =  -2.154619
  a_1（x 的系数）         =   4.909739
  a_2（cos(x) 的系数）    =   2.682703
  a_3（sin(x) 的系数）    =  -1.901834

拟合函数：f(x) = -2.1546 + 4.9097·x + 2.6827·cos(x) + -1.9018·sin(x)

均方根误差（RMS）= 0.529299
均方误差（L）    = 0.280158

==============================================
  各拟合方法均方根误差（RMS）汇总对比
==============================================
拟合方法                                 RMS 误差          均方误差 L        
--------------------------------------------------------------------
多项式 P_1（2 个参数）                       2.155425        4.645856      
多项式 P_2（3 个参数）                       0.553638        0.306515      
多项式 P_3（4 个参数）                       0.375963        0.141348      
多项式 P_4（5 个参数）                       0.365702        0.133738      
混合基 {1, x, cos(x), sin(x)}           0.529299        0.280158      

教材例3.5.3参考值（P_2拟合）：
  a_2=0.7542712, a_1=0.98752785, a_0=0.53554452
  L = 0.17507558（均方误差）
