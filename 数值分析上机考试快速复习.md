## 2. 平方逼近理论与代码

### 2.1 核心思想

**问题**：在函数空间中找 φ*(x)，使得 ||f - φ*||₂ 最小，即：

$$\min_{\varphi \in \Phi} \int_a^b [f(x) - \varphi(x)]^2 dx$$

**解法**：对各系数求偏导令其为 0，得到**法方程组（Gram 方程组）**：

$$G \cdot \mathbf{c} = \mathbf{f}$$

其中：
- **Gram 矩阵** $G_{ij} = (\varphi_i, \varphi_j) = \int_a^b \varphi_i(x)\varphi_j(x) dx$
- **右端向量** $f_j = (f, \varphi_j) = \int_a^b f(x)\varphi_j(x) dx$

### 2.2 两种基函数的对比（最关键！）

| 基函数 | Gram 矩阵 | 条件数 | 数值稳定性 |
|--------|----------|--------|-----------|
| 幂函数 $\varphi_k = x^k$ | Hilbert 矩阵 $H_{ij} = \frac{1}{i+j+1}$ | n=5 时约 $10^{12}$，n=10 时约 $10^{13}$ | **极差** |
| Legendre 正交多项式 | **对角矩阵**，$G_{kk} = \frac{1}{2k+1}$ | **= 1（最优！）** | **完美** |

### 2.3 幂函数基代码

```matlab
f = @(x) exp(x);    % 被逼近函数
n = 5;              % 逼近次数

% ① 构造 Gram 矩阵（Hilbert 矩阵）
% G(i,j) = ∫₀¹ xⁱ⁻¹ · xʲ⁻¹ dx = 1/(i+j-1)  =>  恰好是 Hilbert 矩阵
G = hilb(n+1);      % MATLAB 内置，生成 (n+1)×(n+1) Hilbert 矩阵

% ② 构造右端向量 f_vec(j+1) = ∫₀¹ f(x)·xʲ dx
f_vec = zeros(n+1, 1);
for j = 0:n
    f_vec(j+1) = integral(@(x) exp(x) .* x.^j, 0, 1);
end

% ③ 求解法方程组
c = G \ f_vec;      % c(k+1) 是 x^k 的系数

% ④ 计算条件数
fprintf('条件数 cond(G) = %.4e\n', cond(G));

% ⑤ 在一组点上求逼近多项式的值（Horner 嵌套法）
x_plot = linspace(0, 1, 500);
phi = c(n+1) * ones(size(x_plot));   % 从最高次系数开始
for k = n:-1:1
    phi = phi .* x_plot + c(k);       % 秦九韶（Horner）算法
end

% ⑥ 计算误差
L2_err = sqrt(integral(@(x) (exp(x) - polyval(flipud(c), x)).^2, 0, 1));
```

### 2.4 Legendre 正交基代码

Legendre 多项式 $P_k(t)$ 定义在 $[-1,1]$ 上，变量替换 $t = 2x-1$ 映射到 $[0,1]$。

**三项递推关系**（$P_0=1, P_1=t$，然后递推）：
$$P_{k+1}(t) = \frac{2k+1}{k+1} t \cdot P_k(t) - \frac{k}{k+1} P_{k-1}(t)$$

```matlab
% 计算第 k 次 Legendre 多项式在 [0,1] 上的值（传入 x∈[0,1]）
function vals = legendre_on_01(k, x)
    t = 2*x - 1;          % 变量替换 [0,1] → [-1,1]
    if k == 0
        vals = ones(size(x));
    elseif k == 1
        vals = t;
    else
        p_prev = ones(size(x));
        p_curr = t;
        for j = 1:k-1
            p_next = ((2*j+1)*t.*p_curr - j*p_prev) / (j+1);
            p_prev = p_curr;
            p_curr = p_next;
        end
        vals = p_curr;
    end
end
```

```matlab
f = @(x) exp(x);
n = 5;

% ① Gram 矩阵是对角阵，系数可逐项显式计算
c_leg = zeros(n+1, 1);
for k = 0:n
    inner_fk = integral(@(x) exp(x) .* legendre_on_01(k, x), 0, 1);
    norm_sq = 1 / (2*k+1);   % (φ_k, φ_k) = 1/(2k+1)
    c_leg(k+1) = inner_fk / norm_sq;   % c_k = (f, φ_k) / (φ_k, φ_k)
end

% ② 求逼近多项式的值
phi_leg = zeros(size(x_plot));
for k = 0:n
    phi_leg = phi_leg + c_leg(k+1) * legendre_on_01(k, x_plot);
end

% ③ L2 误差公式（利用 Bessel 不等式，不用积分）
f_L2sq = (exp(2)-1)/2;   % ||f||₂² = ∫₀¹ e²ˣ dx = (e²-1)/2
norms_sq = 1 ./ (2*(0:n)' + 1);
L2_err = sqrt(max(0, f_L2sq - sum(c_leg.^2 .* norms_sq)));
```

## 3. 最小二乘拟合理论与代码

### 3.1 核心思想

**问题**：给定 $m$ 个数据点 $(x_i, y_i)$，找函数 $\varphi(x) = \sum_{k=0}^n c_k \varphi_k(x)$ 使得：

$$L = \sum_{i=1}^m [y_i - \varphi(x_i)]^2 \to \min$$

**与最佳平方逼近的区别**：
- 平方逼近：对连续函数，用积分定义内积
- 最小二乘：对离散数据点，用求和定义内积

### 3.2 矩阵形式

构造**设计矩阵** $A$（$m \times (n+1)$）：$A_{ik} = \varphi_{k-1}(x_i)$

然后解法方程组：$(A^T A) \mathbf{c} = A^T \mathbf{y}$

在 MATLAB 中直接用 `A \ y` 即可（内部用 QR 分解，比先算 $A^T A$ 更稳定）。

### 3.3 多项式最小二乘拟合代码

```matlab
% 数据点
x_data = [0.000, 0.895, 1.641, 2.512, 3.542, 4.054, 4.602, 5.063, 5.354, 5.617];
y_data = [1.000, 1.803, 3.680, 7.320, 13.59, 17.41, 22.19, 24.89, 26.55, 29.77];

m = length(x_data);  % 数据点个数
n = 2;               % 拟合多项式次数

% ① 构造设计矩阵（Vandermonde 型）
% A(i, k+1) = x_i^k，即每列是 x 的一个次幂
A = zeros(m, n+1);
for k = 0:n
    A(:, k+1) = x_data(:).^k;   % .^ 逐元素幂；(:) 确保列向量
end

% ② 求最小二乘解
c = A \ y_data(:);   % 用 QR 分解求解，比 (A'*A)\(A'*y) 更稳定

% ③ 计算误差
y_fit = A * c;                           % 在数据点处的拟合值
rms = sqrt(mean((y_fit - y_data(:)).^2)); % 均方根误差（RMS）

fprintf('RMS 误差 = %.6f\n', rms);
fprintf('条件数 cond(A) = %.4e\n', cond(A));

% ④ 画图
x_plot = linspace(min(x_data), max(x_data), 500);
A_plot = zeros(length(x_plot), n+1);
for k = 0:n
    A_plot(:, k+1) = x_plot(:).^k;
end
y_plot = A_plot * c;

plot(x_data, y_data, 'ro', 'MarkerSize', 8, 'DisplayName', '数据点');
hold on;
plot(x_plot, y_plot, 'b-', 'LineWidth', 1.5, 'DisplayName', '拟合曲线');
legend; grid on;
```

### 3.4 混合基函数拟合代码

```matlab
% 基函数：{1, x, cos(x), sin(x)}
A_mix = [ones(m,1), x_data(:), cos(x_data(:)), sin(x_data(:))];
c_mix = A_mix \ y_data(:);
% 拟合函数：f(x) = c1 + c2·x + c3·cos(x) + c4·sin(x)
fprintf('拟合系数：%.4f + %.4f·x + %.4f·cos(x) + %.4f·sin(x)\n', c_mix);
```

## 4. 数值积分：复合梯形 & 复合Simpson

### 4.1 两种公式对比

| 公式 | 表达式 | 误差阶 | 代数精度 |
|------|--------|--------|---------|
| 复合梯形 | $T_n = \frac{h}{2}[f(a)+f(b)+2\sum_{i=1}^{n-1}f(x_i)]$ | $O(h^2)$ | 1 |
| 复合Simpson | $S_m = \frac{h}{3}[f(a)+f(b)+4\sum_{\text{奇}}+2\sum_{\text{偶}}]$ | $O(h^4)$ | 3 |

> **步长** $h = (b-a)/n$；**Simpson 要求 n 为偶数**，$n=2m$

### 4.2 复合梯形代码

```matlab
f = @(x) 4 ./ (1 + x.^2);   % 被积函数
a = 0; b = 1;                % 积分区间
n = 1000;                    % 子区间数（越大越精确，但有最优步长）

h = (b - a) / n;
x = linspace(a, b, n+1);    % n+1 个等间距节点
fx = f(x);

% 复合梯形公式
T = h/2 * (fx(1) + fx(end) + 2*sum(fx(2:end-1)));
fprintf('复合梯形结果：%.10f\n', T);
```

### 4.3 复合Simpson代码

```matlab
m = 500;    % m 对子区间（总共 2m 等份，共 2m+1 个节点）
h = (b - a) / (2*m);
x = linspace(a, b, 2*m+1);
fx = f(x);

% 节点编号规则（MATLAB 从 1 开始）：
%   端点：fx(1) 和 fx(end)
%   奇数节点（下标 2,4,...,2m）：系数 4（中点类型）
%   偶数内节点（下标 3,5,...,2m-1）：系数 2
S = h/3 * (fx(1) + fx(end) + 4*sum(fx(2:2:end-1)) + 2*sum(fx(3:2:end-2)));
fprintf('复合Simpson结果：%.10f\n', S);
```

### 4.4 最优步长与误差饱和

当步长太小时，误差不再减小！原因：
- 截断误差（离散误差）随 $h$ 减小而减小
- 舍入误差随节点数增多而增大（$\approx O(\text{eps}/h)$）
- 两者平衡给出**最优步长**：
  - 梯形：$h_{\text{opt}} \approx \text{eps}^{1/3} \approx 6\times10^{-6}$
  - Simpson：$h_{\text{opt}} \approx \text{eps}^{1/5} \approx 10^{-3}$

## 5. Romberg 积分法

### 5.1 思想（Richardson 外推加速）

对梯形公式逐次对分，然后做**外推**消去误差主项，大幅提升精度：

$$T_{m,j} = \frac{4^{j-1} \cdot T_{m,j-1} - T_{m-1,j-1}}{4^{j-1} - 1}$$

- $T_{m,1}$：第 $m$ 次对分后的梯形值
- $T_{m,m}$：对角线元素，精度为 $O(h^{2m})$
- 只用很少的函数求值次数就能达到极高精度

### 5.2 变步长梯形递推公式（节省计算的关键）

$$T_{m,1} = \frac{1}{2}T_{m-1,1} + h_m \sum_{\text{新增中点}} f(x_{\text{新}})$$

每次对分只需计算**新增的中点函数值**，而不重新计算已有节点。

### 5.3 Romberg 完整代码

```matlab
f = @(x) 4 ./ (1 + x.^2);
a = 0; b = 1;
pi_exact = pi;

max_level = 10;       % 最大对分次数
tol = 1e-12;          % 收敛容限

T = zeros(max_level, max_level);  % Romberg 表

% 第 1 层：最粗的梯形（仅用两端点）
T(1, 1) = (b-a)/2 * (f(a) + f(b));

for m = 2:max_level
    % ① 计算新增中点，更新第一列（变步长梯形递推）
    h_prev = (b-a) / 2^(m-2);     % 上一层步长
    h_cur  = (b-a) / 2^(m-1);     % 当前层步长（= h_prev/2）
    k_new  = 1 : 2^(m-2);         % 新增中点编号
    x_new  = a + (k_new - 0.5) * h_prev;  % 新增中点坐标
    T(m, 1) = 0.5*T(m-1,1) + h_cur * sum(f(x_new));

    % ② Richardson 外推填充各列
    for j = 2:m
        coeff = 4^(j-1);
        T(m, j) = (coeff * T(m,j-1) - T(m-1,j-1)) / (coeff - 1);
    end

    fprintf('m=%d: T_{m,m} = %.15f, 误差 = %.4e\n', m, T(m,m), abs(T(m,m)-pi_exact));

    % ③ 收敛判断
    if abs(T(m,m) - T(m-1,m-1)) < tol
        fprintf('收敛于第 %d 层！\n', m);
        break;
    end
end

fprintf('Romberg 结果：%.15f\n', T(m,m));
```

## 6. 自适应 Simpson 积分法

### 6.1 思想

不用均匀步长，而是**在函数变化剧烈处自动细分**，在平缓处保持较大步长。

**局部误差估计**：

$$|误差| \approx \frac{|S(a,m) + S(m,b) - S(a,b)|}{15}$$

（$m=(a+b)/2$ 为中点，$15 = 2^4 - 1$ 来自 Simpson 误差阶 $O(h^5)$，步长减半误差缩小 $2^5=32$ 倍）

若误差满足容限则接受，否则**递归细化**。

### 6.2 自适应Simpson代码（含辅助函数）

```matlab
% 主程序调用
f = @(x) 4 ./ (1 + x.^2);
a = 0; b = 1;
tol = 1e-10;

S_init = simp1(f, a, b);   % 初始整区间 Simpson 值
[result, cnt] = adapt_simp(f, a, b, tol, S_init);
fprintf('自适应Simpson结果：%.15f，函数求值 %d 次\n', result, cnt);

% =========================================================
% 辅助函数（放在文件末尾）

function S = simp1(f, a, b)
    % 单区间 Simpson 公式：(b-a)/6 * [f(a) + 4f(m) + f(b)]
    m = (a + b) / 2;
    S = (b-a)/6 * (f(a) + 4*f(m) + f(b));
end

function [I, cnt] = adapt_simp(f, a, b, tol, S_ab)
    % 自适应 Simpson 积分（递归）
    m = (a + b) / 2;
    S_am = simp1(f, a, m);
    S_mb = simp1(f, m, b);
    S2 = S_am + S_mb;
    cnt = 3;   % 新增3个函数求值点

    if abs(S2 - S_ab) / 15 <= tol
        I = S2 + (S2 - S_ab) / 15;   % 加 Richardson 校正
    else
        [I_l, c_l] = adapt_simp(f, a, m, tol/2, S_am);  % 递归细化左半
        [I_r, c_r] = adapt_simp(f, m, b, tol/2, S_mb);  % 递归细化右半
        I = I_l + I_r;
        cnt = cnt + c_l + c_r;
    end
end
```

## 7. 蒙特卡罗积分

### 7.1 两种方法

#### 方法一：随机投点法（Hit-or-Miss）

在矩形区域内均匀撒 $N$ 个随机点，统计落在曲线下方的比例：

$$I \approx \text{面积} \times \frac{\text{命中次数}}{N}$$

```matlab
f = @(x) exp(-x);
a = 0; b = 1;
N = 1e6;
f_max = 1;   % f(x) 在 [a,b] 上的最大值

% 生成 N 个均匀随机点
x_rand = a + (b-a) * rand(1, N);   % rand(1,N)：N 个 [0,1] 均匀随机数
y_rand = f_max * rand(1, N);

% 统计落在曲线下的点
hits = sum(y_rand <= f(x_rand));    % sum：统计 true 的个数

I_mc = (b-a) * f_max * hits / N;
fprintf('投点法 MC 结果：%.6f（精确值 %.6f）\n', I_mc, 1-exp(-1));
```

#### 方法二：样本均值法（Sample Mean，更高效）

$$I = \int_a^b f(x) dx \approx (b-a) \cdot \frac{1}{N} \sum_{i=1}^N f(x_i)$$

```matlab
N = 1e6;
x_rand = a + (b-a) * rand(1, N);
I_mc = (b-a) * mean(f(x_rand));    % mean：求均值
fprintf('均值法 MC 结果：%.6f\n', I_mc);
```

**样本均值法的标准差比投点法小**，效率更高（约 6 倍）！

### 7.2 蒲丰投针估计 π

```matlab
N = 1e6;
l = 1; d = 2;   % 针长 l，条纹间距 d（要求 l ≤ d）

% 随机生成针的位置和角度
x_center = d * rand(1, N);         % 针中心到最近直线的距离
theta = pi/2 * rand(1, N);         % 针与直线夹角（0 到 π/2 即可）

% 针相交条件：针的投影长度 > 到最近直线的距离
cross = x_center <= l/2 * sin(theta);
p_hat = sum(cross) / N;            % 相交概率

% 由理论概率 P = 2l/(π·d) 反推 π
pi_est = 2 * l / (d * p_hat);
fprintf('蒲丰投针估计 π = %.6f（误差 %.4e）\n', pi_est, abs(pi_est-pi));
```

## 8. 无穷积分的处理技巧

### 8.1 方法一：截断上限

当 $f(x) \to 0$（$x \to \infty$）很快时，选取足够大的截断点 $X$：

```matlab
% 计算 ∫₀^∞ x³/(eˣ+1) dx
% 当 x=50 时，f(50) ≈ 0（可忽略），截断为 [1e-10, 50]
x_min = 1e-10;  x_max = 50;
I = integral(f, x_min, x_max);

% 或用 MATLAB 直接算广义积分
I = integral(f, 0, Inf);
```

### 8.2 方法二：变量替换

令 $t = e^{-x}$，将 $[0,\infty)$ 变换为 $[0,1]$：

$$\int_0^{\infty} f(x)dx = \int_0^1 f(-\ln t) \cdot \frac{1}{t} dt$$

```matlab
% 例：∫₀^∞ x³/(eˣ+1) dx
% 令 t = e^{-x}, x = -ln(t), dx = -1/t dt
% => ∫₀¹ (-ln t)³ / (1+t) dt
g = @(t) (-log(t)).^3 ./ (1 + t);

m = 50000;
h = 1/(2*m);
t_vals = linspace(1e-12, 1-1e-12, 2*m+1);  % 避开端点奇点
gt = g(t_vals);
I = h/3 * (gt(1)+gt(end) + 4*sum(gt(2:2:end-1)) + 2*sum(gt(3:2:end-2)));
```

### 8.3 稳健函数处理数值奇点

```matlab
function y = planck_func(x)
    % 稳健处理 f(x) = x³/(eˣ+1)
    if x <= 0
        y = 0;            % 极限值
    elseif x > 500
        y = 0;            % 接近 0，可忽略
    elseif x > 37         % exp(x) 将溢出
        y = x^3 * exp(-x);  % 用近似避免 Inf
    else
        y = x^3 / (exp(x) + 1);  % 正常计算
    end
end
```

## 9. 考试核心公式汇总

### 平方逼近

| 量 | 公式 |
|----|------|
| 内积 | $(f,g) = \int_a^b f(x)g(x)dx$ |
| Gram 矩阵 | $G_{ij} = (\varphi_i, \varphi_j)$ |
| 法方程组 | $G\mathbf{c} = \mathbf{f}$，$f_j = (f, \varphi_j)$ |
| Hilbert 矩阵 | $H_{ij} = \frac{1}{i+j-1}$（幂函数基 $\Rightarrow$ 极度病态） |
| Legendre 正交 | $\int_0^1 P_i(2x-1)P_j(2x-1)dx = \frac{\delta_{ij}}{2i+1}$（对角 Gram）|
| Legendre 系数 | $c_k = (2k+1)\int_0^1 f(x)P_k(2x-1)dx$ |
| L2 误差公式 | $\|f-\varphi^*\|_2^2 = \|f\|_2^2 - \sum_{k=0}^n c_k^2 \cdot \frac{1}{2k+1}$ |
| Legendre 递推 | $P_0=1,\ P_1=t,\ P_{k+1}=\frac{(2k+1)t P_k - k P_{k-1}}{k+1}$ |

### 数值积分

| 方法 | 公式 | 误差阶 |
|------|------|--------|
| 复合梯形 | $T_n = \frac{h}{2}[f(a)+f(b)+2\sum_{i=1}^{n-1}f(x_i)]$ | $O(h^2)$ |
| 复合Simpson | $S_m = \frac{h}{3}[f(a)+f(b)+4\sum_{\text{奇}}+2\sum_{\text{偶}}]$，$h=(b-a)/(2m)$ | $O(h^4)$ |
| Romberg 外推 | $T_{m,j} = \frac{4^{j-1}T_{m,j-1} - T_{m-1,j-1}}{4^{j-1}-1}$ | $O(h^{2m})$ |
| 自适应误差估计 | $\text{err} \approx \|S(a,m)+S(m,b) - S(a,b)\| / 15$ | — |
| 投点法MC | $I \approx (b-a)f_{\max} \cdot N_{\text{命中}}/N$ | $O(1/\sqrt{N})$ |
| 均值法MC | $I \approx (b-a) \cdot \frac{1}{N}\sum f(x_i)$ | $O(1/\sqrt{N})$ |

## 10. 完整代码模板

### 模板1：最佳平方逼近（通用）

```matlab
clear; close all; clc;

f = @(x) exp(x);   % 修改这里的被逼近函数
a = 0; b = 1;       % 修改积分区间
n = 5;              % 逼近次数

%% 方法A：幂函数基（Hilbert矩阵，病态）
G = hilb(n+1);
f_vec = zeros(n+1, 1);
for j = 0:n
    f_vec(j+1) = integral(@(x) f(x) .* x.^j, a, b);
end
c = G \ f_vec;
fprintf('幂函数基 cond(G) = %.4e\n', cond(G));

%% 方法B：Legendre 正交基（稳定）
c_leg = zeros(n+1, 1);
for k = 0:n
    inner = integral(@(x) f(x) .* legendre_on_01(k, x), a, b);
    c_leg(k+1) = (2*k+1) * inner;
end
fprintf('Legendre 基 cond(G) = 1（正交）\n');

%% 绘图对比
x_plot = linspace(a, b, 500);
y_exact = f(x_plot);

% 幂函数基在 x_plot 上的值（Horner 法）
phi_mono = c(n+1)*ones(size(x_plot));
for k = n:-1:1
    phi_mono = phi_mono .* x_plot + c(k);
end

% Legendre 基在 x_plot 上的值
phi_leg = zeros(size(x_plot));
for k = 0:n
    phi_leg = phi_leg + c_leg(k+1) * legendre_on_01(k, x_plot);
end

figure;
plot(x_plot, y_exact, 'k-', 'LineWidth', 2, 'DisplayName', '精确 f(x)');
hold on;
plot(x_plot, phi_mono, 'b--', 'LineWidth', 1.5, 'DisplayName', '幂函数基逼近');
plot(x_plot, phi_leg,  'r-.', 'LineWidth', 1.5, 'DisplayName', 'Legendre基逼近');
legend; grid on; title('最佳平方逼近对比');

% Legendre 函数（放在文件末尾）
function vals = legendre_on_01(k, x)
    t = 2*x - 1;
    if k == 0; vals = ones(size(x)); return; end
    if k == 1; vals = t; return; end
    p0 = ones(size(x)); p1 = t;
    for j = 1:k-1
        p2 = ((2*j+1)*t.*p1 - j*p0)/(j+1);
        p0 = p1; p1 = p2;
    end
    vals = p1;
end
```

### 模板2：三种积分法一起算（通用）

```matlab
clear; close all; clc;

f = @(x) 4./(1+x.^2);   % 修改这里
a = 0; b = 1;             % 修改积分区间
I_exact = pi;             % 修改精确值（用 integral 估算）

%% 复合梯形
n = 1000;
h = (b-a)/n;
x = linspace(a, b, n+1);
fx = f(x);
T = h/2 * (fx(1)+fx(end) + 2*sum(fx(2:end-1)));
fprintf('复合梯形：%.10f  误差 %.4e\n', T, abs(T-I_exact));

%% 复合Simpson
m = 500;
h = (b-a)/(2*m);
x = linspace(a, b, 2*m+1);
fx = f(x);
S = h/3 * (fx(1)+fx(end) + 4*sum(fx(2:2:end-1)) + 2*sum(fx(3:2:end-2)));
fprintf('复合Simpson：%.10f  误差 %.4e\n', S, abs(S-I_exact));

%% Romberg
max_lv = 10; tol = 1e-12;
T_rom = zeros(max_lv, max_lv);
T_rom(1,1) = (b-a)/2*(f(a)+f(b));
for m = 2:max_lv
    h_p = (b-a)/2^(m-2); h_c = (b-a)/2^(m-1);
    x_new = a + ((1:2^(m-2)) - 0.5)*h_p;
    T_rom(m,1) = 0.5*T_rom(m-1,1) + h_c*sum(f(x_new));
    for j = 2:m
        c4 = 4^(j-1);
        T_rom(m,j) = (c4*T_rom(m,j-1) - T_rom(m-1,j-1))/(c4-1);
    end
    if abs(T_rom(m,m)-T_rom(m-1,m-1)) < tol; break; end
end
fprintf('Romberg：    %.10f  误差 %.4e\n', T_rom(m,m), abs(T_rom(m,m)-I_exact));

%% 自适应Simpson
S0 = (b-a)/6*(f(a)+4*f((a+b)/2)+f(b));
[I_ad, ~] = adapt_simp(f, a, b, 1e-10, S0);
fprintf('自适应：     %.10f  误差 %.4e\n', I_ad, abs(I_ad-I_exact));

%% MATLAB 内置（参考）
I_ref = integral(f, a, b);
fprintf('MATLAB内置： %.10f  误差 %.4e\n', I_ref, abs(I_ref-I_exact));

% --- 辅助函数（文件末尾）---
function S = simp1(f, a, b)
    m = (a+b)/2;
    S = (b-a)/6*(f(a)+4*f(m)+f(b));
end

function [I, cnt] = adapt_simp(f, a, b, tol, S_ab)
    m = (a+b)/2;
    S_am = simp1(f, a, m);
    S_mb = simp1(f, m, b);
    S2 = S_am + S_mb;
    cnt = 3;
    if abs(S2-S_ab)/15 <= tol
        I = S2 + (S2-S_ab)/15;
    else
        [Il,cl] = adapt_simp(f, a, m, tol/2, S_am);
        [Ir,cr] = adapt_simp(f, m, b, tol/2, S_mb);
        I = Il+Ir; cnt = cnt+cl+cr;
    end
end
```

## 附录：常见错误与注意事项

### 错误1：忘记逐元素运算符

```matlab
% 错误：
f = @(x) 4 / (1 + x^2);    % 当 x 是向量时会出错！

% 正确：
f = @(x) 4 ./ (1 + x.^2);  % 逐元素除法和幂
```

### 错误2：Simpson 下标搞错

```matlab
% Simpson 节点 x₀, x₁, x₂, ..., x_{2m}，共 2m+1 个点
% MATLAB 下标从 1 开始，所以：
% x₀ 是 fx(1)，x₁ 是 fx(2)，x_{2m} 是 fx(end)
% 系数 4 的节点：x₁, x₃, ..., x_{2m-1}  =>  fx(2:2:end-1)
% 系数 2 的节点：x₂, x₄, ..., x_{2m-2}  =>  fx(3:2:end-2)
```

### 错误3：Romberg 对分层的新增点

```matlab
% 第 m 层，当前步长 h_cur = (b-a)/2^(m-1)
% 新增的中点在上一层步长 h_prev = (b-a)/2^(m-2) 的各区间中点
% x_new = a + (k - 0.5) * h_prev，k = 1, 2, ..., 2^(m-2)
```

### 错误4：最小二乘设计矩阵的构造

```matlab
% A(i, k+1) = x_i^k（行是数据点，列是基函数）
% 共 m 行（数据点数），n+1 列（基函数数）
% 当 m > n+1 时是超定方程组，\ 自动求最小二乘解
```

### 注意：arrayfun vs 向量化

```matlab
% 当函数内部有 if 语句（如 planck_func），不能直接向量输入
fx = arrayfun(planck_func, x);   % 对 x 的每个元素单独调用函数

% 普通匿名函数支持向量直接输入（用 .运算符）
fx = f(x);   % 直接向量化
```
